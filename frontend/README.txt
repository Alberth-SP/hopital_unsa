Para empezar el proyecto usar "npm install" en la terminal...
se recomienda usar la terminal de Visual ctrl+ñ, al empezar 
agregar un componente dependiendo al modulo que corresponda,
no olvidar rutear sus componentes en app-routing.module.ts 
para que se muestre en la interfaz, se estara usando primeng
para facilitar el trabajo y fontawesome que es para usar iconos
todo esto de por si ya estara instalado, solo tienen que usarlo
 y finalmente solo hagan cambios en sus modulos ya que solo 
trabajran en sus vistas.

Los valores de la lista del menu vertical esta en menu-nav.component
si desean agregar alguno lo hacen ahi no hice ruteos de nada mas que
del home por ahora, es solo un avance aun falta darle detalles pero
se ve bien creo para ser el primer prototipo, si desean cambiar la 
interfaz haganlo el el archivo site.css si es algun componente de 
primeng y si no usen el css de cada componente.

Aun no se haran servicios solo creen sus componentes y muestren las 
vistas en vezde la tabla.

es poco estetico :v.
